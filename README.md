# Minhas Finanças 💰

Aplicação web para **controle financeiro pessoal**, desenvolvida com foco em simplicidade, organização e visualização clara das despesas.

O sistema permite registrar gastos pontuais e recorrentes, acompanhar o saldo disponível em tempo real e visualizar as movimentações do mês de forma prática.

---

## 📌 Funcionalidades

* ✅ Cadastro de **despesas pontuais**
* 🔁 Controle de **despesas recorrentes**
* 📅 Organização por **mês**
* 📊 Visualização de **saldo disponível (hoje)**
* 📈 **Projeção financeira total**
* 🏷️ Categorização de despesas
* 🧾 Histórico de movimentações mensais

---

## 🖥️ Interface

A interface foi pensada para ser limpa e intuitiva, facilitando o uso no dia a dia:

* Cards com informações principais (saldo e projeção)
* Formulário simples para novos lançamentos
* Separação por abas: **Geral**, **Meses** e **Recorrências**

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🚀 Objetivo do Projeto

Este projeto foi desenvolvido com fins de **aprendizado e portfólio**, aplicando conceitos de organização financeira, lógica de programação e boas práticas de interface.

Também serve como base para futuras evoluções, como:

* Autenticação de usuários
* Persistência de dados (banco de dados)
* Gráficos financeiros
* Exportação de relatórios
